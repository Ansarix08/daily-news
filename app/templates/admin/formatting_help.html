Content Formatting Guide

Text Formatting

Basic Formatting

  • Bold: Select text and click the bold button (B) or use Ctrl/Cmd + B
  • Italic: Select text and click the italic button (I) or use Ctrl/Cmd + I
  • Underline: Select text and click the underline button (U) or use Ctrl/Cmd + U
  • Highlight: Select text and click the highlight button

Headings

Use headings to organize your content hierarchically:

  • Heading 1

  • Heading 2

  • Heading 3

  • Heading 4

Lists

Bulleted List:

  • Item 1
  • Item 2
  • Item 3

Numbered List:

  1. First item
  2. Second item
  3. Third item

Links and Media

  • To add a link: Select text and click the link button or use Ctrl/Cmd + K
  • To add an image: Click the image button and upload or provide an image URL
  • To add a table: Click the table button and select the dimensions

Advanced Features

  • Use the code block button for formatting code snippets
  • Use blockquotes for quoted content
  • Adjust font size and family using the style dropdown
  • Use custom colors for text and background

Keyboard Shortcuts

Formatting
  • Bold: Ctrl/Cmd + B
  • Italic: Ctrl/Cmd + I
  • Underline: Ctrl/Cmd + U
  • Link: Ctrl/Cmd + K
Actions
  • Save: Ctrl/Cmd + S
  • Undo: Ctrl/Cmd + Z
  • Redo: Ctrl/Cmd + Y
  • Select All: Ctrl/Cmd + A

Tips & Best Practices

  • Use Headings Properly

    Structure your content with headings (H1-H4) to improve readability and SEO.

  • Optimize Images

    Compress images before uploading to improve load times.

  • Preview Content

    Always preview your content before publishing to ensure proper formatting.

  • Save Regularly

    Use Ctrl/Cmd + S to save your work frequently.
