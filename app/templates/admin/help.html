Admin Dashboard Formatting Guide

1. Basic Text Formatting

Feature How to Use Keyboard Shortcut
Bold Click the B button or select text Ctrl + B
Italic Click the I button or select text Ctrl + I
Underline Click the U button or select text Ctrl + U
Strikethrough Click the S button or select text Alt + Shift + 5

2. Headings and Paragraphs

  • Headings (H1-H6):
    • Click the 'Paragraph' dropdown in toolbar
    • Select desired heading level
    • Or use keyboard: Ctrl + Alt + 1 (for H1) through 6 (for H6)
  • Alignment:
    • Use alignment buttons in toolbar (Left, Center, Right, Justify)
    • Or select text and use Ctrl + Shift + L/C/R/J

3. Lists and Indentation

Bullet Lists:
  • Click bullet list icon
  • Or press Ctrl + Shift + 8
  • Press Tab to indent
Numbered Lists:
  • Click numbered list icon
  • Or press Ctrl + Shift + 7
  • Press Tab to indent

4. Tables

  1. Click the table icon in toolbar
  2. Drag to select rows and columns
  3. Or click "Table" â†’ "Insert table"
  4. Table editing:
    • Right-click for cell options
    • Use toolbar for adding/deleting rows/columns
    • Drag cell borders to resize

5. Special Features

Code Blocks:
  1. Click the code block icon
  2. Select language
  3. Paste or type code
Blockquotes:
  1. Click blockquote icon
  2. Or press Ctrl + Shift + Q
  3. Type or paste quote

6. Images and Media

Adding Images:
  1. Click the image icon
  2. Choose:
    • Upload: Select file from computer
    • URL: Paste image URL
  3. Resize: Drag image corners
  4. Edit: Double-click image for options

7. Quick Tips

  • Select text to see quick formatting toolbar
  • Right-click for context menu with more options
  • Use Ctrl + Z to undo and Ctrl + Y to redo
  • Press F11 for fullscreen editing
  • Use Ctrl + S to save your work
